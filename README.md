# Memo
projet Memo apple

POC Apple natif pour apprendre :

- SwiftUI
- SwiftData
- CloudKit

## Objectif V0

Créer une application simple permettant de gérer des personnes :

- nom
- notes
- date de création

## Architecture cible

- SwiftUI pour l’interface
- SwiftData pour le stockage local
- CloudKit pour la synchronisation iCloud

## Règle projet

Pas de fonctionnalité avancée avant validation du POC SwiftData + CloudKit.
